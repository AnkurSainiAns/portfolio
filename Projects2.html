Informative College Website

Link: Tech Pluse - Engineering College Pilani

Project Overview

The Engineering College Pilani website is a static, informative portal built to showcase college-related details such as academic departments, course offerings, admissions, faculty, and contact information. Designed with clarity and professionalism, the website ensures that visitors—including students and parents—can easily find essential information about the college.

Key Features

  • Responsive Design: Optimized for both desktop and mobile devices.
  • Product Showcase: High-quality images and detailed descriptions of bridal wear.
  • Homepage: Welcoming design with college branding
  • About Section: Brief overview of the institution
  • Admissions Page: Guidance on how to apply
  • Contact Form/Page: Easy contact option for visitors
  • Fast Loading: Lightweight code, static hosting

Technologies Used

  • WordPress : CMS
  • WooCommerce : E-commerce plugin for WordPress
  • Free Hosting via InfinityFree

My Role

  • implemented SEO strategies to improve product visibility.

  • Uploaded static content and tested cross-device compatibility
